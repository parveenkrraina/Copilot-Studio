## üìù Step-by-Step Instructions

### Step 1: Accessing Copilot Studio

1. Open your web browser and navigate to [Copilot Studio](https://copilotstudio.microsoft.com)
2. Sign in with your Microsoft account
3. If this is your first time, you may need to:
   - Accept the terms of service
   - Select your default environment
   - Configure basic settings

**Expected Result:** You should see the Copilot Studio home page with options to create a new copilot.

### Step 2: Understanding the Interface

Take a moment to familiarize yourself with the main components:

1. **Navigation Panel (Left Side)**
   - Home: Main dashboard
   - Copilots: List of your copilots
   - Create: Quick access to create new copilots
   - Settings: Environment and user settings

2. **Main Canvas (Center)**
   - Where you'll design and build your copilot
   - Displays copilot conversation flows
   - Testing interface

3. **Properties Panel (Right Side)**
   - Configuration options
   - Entity settings
   - Variable definitions

4. **Top Menu Bar**
   - Save functionality
   - Test copilot button
   - Publish options
   - Help and documentation

### Step 3: Creating Your First Copilot

1. Click on **"Create"** in the left navigation panel
2. Select **"New copilot"**
3. You'll see several options:
   - **Try the conversational boosting option**: Uses AI to enhance responses
   - **Start from a template**: Pre-built copilots for common scenarios
   - **Start from blank**: Build from scratch

4. For this lab, select **"Start from blank"**

### Step 4: Configuring Your Copilot

1. **Name your copilot**: Enter a descriptive name (e.g., "My First Copilot")
2. **Choose a language**: Select your preferred language
3. **Select an environment**: Choose your default environment
4. **Add a description** (optional): Brief description of your copilot's purpose

5. Click **"Create"** button

**Expected Result:** Your copilot is created and you're taken to the authoring canvas.

### Step 5: Exploring the Authoring Canvas

After creation, you'll see:

1. **Topics Panel**: Pre-built system topics
   - Greeting
   - Goodbye
   - Fallback
   - Escalate

2. **Conversation Start**: The entry point for users

3. **Test Chat Panel**: Click "Test your copilot" to open the test interface

### Step 6: Testing Your Copilot

1. Click **"Test your copilot"** button in the bottom-left corner
2. The test chat window opens
3. Try typing: "Hello"
4. Observe the default greeting response
5. Try typing: "Goodbye"
6. See how the copilot responds to different inputs

**Expected Result:** Your copilot responds with default greetings and responses.

### Step 7: Viewing Copilot Settings

1. Click on **"Settings"** in the top-right corner
2. Explore the available settings:
   - **General**: Basic copilot information
   - **Security**: Authentication settings
   - **Data**: Data handling preferences
   - **Channels**: Where your copilot can be deployed
   - **AI Capabilities**: AI features configuration

3. Take note of the settings but don't change anything yet

## üîç Troubleshooting

**Issue**: Cannot access Copilot Studio
- **Solution**: Ensure you have the correct license or trial access. Check with your administrator.

**Issue**: Environment not available
- **Solution**: You may need to create or be granted access to an environment. Contact your Power Platform administrator.

**Issue**: Copilot not responding in test chat
- **Solution**: Click the "Reset" button in the test chat to restart the conversation.
